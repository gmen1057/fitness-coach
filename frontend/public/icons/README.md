# PWA Icons

Place the following icons in this directory:

- `icon-192.png` - 192x192px - Required for PWA install prompt
- `icon-512.png` - 512x512px - Required for splash screen
- `workout-96.png` - 96x96px - Shortcut icon for workout page
- `chat-96.png` - 96x96px - Shortcut icon for chat page

You can generate these icons from a single source image using:
- https://favicon.io/
- https://realfavicongenerator.net/
- ImageMagick: `convert source.png -resize 192x192 icon-192.png`

Recommended design: Simple dumbbell or fitness icon with emerald/green theme color.
